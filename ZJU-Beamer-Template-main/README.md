## 使用说明
1. 下载并安装 TeXLive；
2. 下载本模板文件；
3. 编写 ```.tex``` 文件；
4. 使用 ```XeLaTeX``` 编译（当前仅支持 ```XeLaTeX``` 方式编译，已在 ```Win11 +
TeXLive 2022```和 ```Ubuntu 22.04 + TeXLive 2022``` 平台上测试）。


此模板大量参考了[华中师范大学beamer](https://github.com/K-JW/CCNU_BeamerTemplate)模板，在此向原作者表示感谢。
